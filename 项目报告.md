# 3D打印耗材烘干箱项目报告

## 项目概述

本项目是一个基于ESP32-S3的智能3D打印耗材烘干箱控制系统，旨在解决3D打印耗材（如PLA、ABS、PETG等）因吸湿导致的打印质量问题。系统通过PTC加热片和风扇实现温湿度控制，结合PID算法实现精准闭环控温，支持本地按键、串口命令和网页控制三种交互方式。

## 项目背景与意义

### 问题背景
3D打印耗材（尤其是PLA、PETG等）在潮湿环境中容易吸收水分，导致打印时出现气泡、拉丝、层间结合力下降等问题，严重影响打印质量。传统烘干方法如烤箱烘干存在温度控制不精确、安全性差等问题。

### 项目意义
1. **精准控温**：采用PID算法实现±1℃的温控精度，避免温度过高损坏耗材
2. **智能监测**：实时监测温湿度变化，提供数据记录和分析功能
3. **多模式控制**：支持本地操作、远程网页控制和自动化预设
4. **安全可靠**：具备多重安全保护机制，包括传感器故障检测、超温保护等

## 核心功能实现

### 1. 温湿度监测与控制
- **传感器**：AHT10高精度温湿度传感器（I2C接口）
- **采样频率**：1Hz实时采样
- **精度**：温度±0.3℃，湿度±2%RH
- **控制算法**：PID闭环控制，支持自动校准

### 2. 多模式人机交互
#### (1) 本地按键操作
- 五向按键（上/下/左/右/中）实现完整菜单导航
- 主界面显示实时温湿度、目标温度、剩余时间
- 菜单功能：启动/停止、参数设置、材料预设、PID校准等

#### (2) 网页控制界面
- 响应式设计，支持PC和移动端访问
- 实时数据曲线展示（温度、湿度、加热功率、风扇转速）
- 远程参数配置和控制指令下发
- 开发人员模式（隐藏入口，支持模拟串口命令）

#### (3) 串口命令行
- 提供丰富的调试和控制命令
- 支持模拟按键、状态查询、故障清除等操作
- 便于自动化测试和集成

### 3. 智能网络管理
- **双模式网络**：
  - STA模式：连接现有Wi-Fi网络
  - AP模式：设备自身作为热点（SSID: Dryer-Config）
- **智能切换**：STA连接失败时自动切换至AP模式
- **配置保存**：Wi-Fi配置持久化存储，断电不丢失

### 4. 安全保护机制
- **传感器故障检测**：AHT10通信异常时自动停止加热
- **超温保护**：温度超过安全阈值时立即切断加热
- **断电恢复**：运行状态自动保存，上电后可继续任务
- **多重故障清除**：支持本地、串口、网页三种方式清除故障

### 5. 材料预设系统
- 内置常见耗材烘干参数：
  - PLA：50℃, 4-6小时
  - ABS：70℃, 4-6小时  
  - PETG：55℃, 6-8小时
  - TPU：45℃, 8-12小时
  - Nylon：70℃, 12-24小时
- 支持用户自定义参数保存和调用

## 技术创新点

### 1. PID自动校准技术
- 采用继电器振荡法（Relay Feedback Method）
- 自动识别系统动态特性，计算最优PID参数
- 无需人工调试，降低使用门槛

### 2. 状态持久化设计
- 运行状态（目标温度、剩余时间、PID参数）实时保存
- 意外断电后恢复运行，保证任务连续性
- 配置数据采用JSON格式存储，易于扩展

### 3. 模块化软件架构
- 核心控制逻辑与用户界面分离
- 支持热插拔式功能模块扩展
- 清晰的API接口设计，便于二次开发

### 4. 多层级故障处理
- 硬件层：MOSFET驱动保护电路
- 软件层：看门狗定时器、堆栈溢出检测
- 应用层：用户可操作的故障清除机制

### 关键算法
```cpp
// PID控制核心算法
double PID::compute(double input, double setpoint) {
    double error = setpoint - input;
    double dInput = input - lastInput;
    
    // 积分项抗饱和
    if (!limitIntegral || (outputSum > outMin && outputSum < outMax)) {
        outputSum += ki * error;
    }
    
    double output = kp * error + outputSum - kd * dInput;
    
    // 输出限幅
    if (output > outMax) output = outMax;
    if (output < outMin) output = outMin;
    
    lastInput = input;
    return output;
}
```

### 数据记录系统
- 日志格式：CSV文件，包含时间戳、温度、湿度、控制量等
- 存储位置：SPIFFS文件系统的`/dryer_log.csv`
- 记录频率：每分钟一条记录，可配置
- 数据字段：
  - `ms`：毫秒时间戳
  - `temp_c`：当前温度(℃)
  - `humidity`：当前湿度(%RH)
  - `target_c`：目标温度(℃)
  - `heater_pct`：加热功率百分比
  - `fan_pct`：风扇转速百分比
  - `active`：运行状态
  - `fault`：故障代码
  - `preset`：材料预设名称
  - `remaining_s`：剩余时间(秒)
